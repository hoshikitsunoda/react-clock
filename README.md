# react-clock
Recreate world clock using React.
# Usage
```
$ node index.js
```
# Uses
* HTML5
* CSS3
* Modern JavaScript (ES2015+)
* React
* Babel
* Moment
* Moment Timezone
* MongoDB
* Express
* Node.js
